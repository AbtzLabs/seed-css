Grid

Grids are used to define columns.
The grid system is responsive, which means you can define the number of columns according to the screen size.

.sm-12.md-6.lg-3
.sm-12.md-6.lg-3
.sm-12.md-6.lg-3
.sm-12.md-6.lg-3
<div class="grid" >
  <div class="row" >
    <div class="col sm-12 md-6" ></div>
    <div class="col sm-12 md-6" ></div>
  </div>
</div>

Offsets are also available.
To apply offsets, just add the sufix -offset to your classname.

.col.sm-12.md-6
.col.sm-12.md-6.md-6-offset
<div class="grid" >
  <div class="row" >
    <div class="col sm-6" ></div>
    <div class="col sm-6 sm-6-offset" ></div>
  </div>
</div>
